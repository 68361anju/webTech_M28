Pseudo class selectors

pseudo class selectors are used to target the state of the specified element.

This pseudo class selectors are of few ways:They are

  • link pseudo class selector
  • visited pseudo class selector
  • Active pseudo class selector
  • hover pseudo class selector
  • focus pseudo class selector
  • checked pseudo class selector
  • first_child pseudo classs selector
  • last_child pseudo classs selector
  • nth _child pseudo classs selector
  • first_of_type pseudo classs selector
  • last_of_type pseudo classs selector
  • nth_of_type pseudo classs selector
